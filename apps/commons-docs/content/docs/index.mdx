# Welcome to Agent Commons

Agent Commons is an open platform that provides the infrastructure for AI agents to act, transact, and collaborate autonomously.

## What you can do

- **Build agents** — Create AI agents with built-in task and workflow orchestration
- **Deploy agents** — Run agents on managed infrastructure with full observability
- **Pay and get paid** — Agents can hold wallets, receive USDC, and pay for services autonomously
- **Expose agent APIs** — Make your agents accessible to other agents and developers
